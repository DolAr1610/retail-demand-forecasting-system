About this system

Demand Forecast MVP is a documentation-first dashboard for a retail demand forecasting project. It helps present the workflow and results in a clean, understandable way — without technical overload.

What it is

Purpose

The system supports a typical forecasting workflow: define a stable dataset window, build a compact subset of products, train several models (from simple baselines to tree-based ML), and compare their quality using clear metrics. The interface is structured so it can be easily used in a thesis, documentation, or a demo presentation where readability matters.

Who it is for

Audience
  • Students & researchers — to demonstrate methodology and model comparison.
  • Analysts — to confirm that models improve over baselines.
  • Stakeholders — to understand results without deep technical details.

What you can show

Deliverables
  • Dataset window + subset definition (store, families, items).
  • Model list (baselines + ML models).
  • Metrics table and compact charts in a “presentation-ready” format.
  • A prediction page that can be connected to a backend later.
